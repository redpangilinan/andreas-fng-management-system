## Overview

### Features (Outdated)
- Login and Registration
- Page Access Validation - Prevents users to access admin only pages
- Sessions and Cookies - Auto Login and Logout
- Full CRUD Operations
- Live Data Tables with AJAX
- Live Search with AJAX
- Responsive UI
